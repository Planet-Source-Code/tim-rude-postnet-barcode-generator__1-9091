<img src="PIC20006202236143004.gif">

### Description

BAS module that allows you to print

PostNet (aka 3 of 5) barcodes used

by the US Postal Service. Can output

to printer or screen (i.e. form,

picturebox, etc.) No DLL/OCX/TTF used.

All code.
